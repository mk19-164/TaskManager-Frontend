# TaskManager Frontend

Frontend de la aplicación **TaskManager**, desarrollado con **React** y **Vite**.

La aplicación permite registrarse, verificar la cuenta por correo, iniciar sesión y gestionar tareas y categorías desde una interfaz oscura, moderna y responsiva.

---

## Tecnologías utilizadas

- React
- Vite
- React Router DOM
- Axios
- CSS

---

## Funcionalidades principales

- Registro de usuarios
- Verificación de correo electrónico
- Login con JWT
- Logout
- Navbar dinámica según sesión iniciada
- Rutas protegidas en frontend
- Dashboard con métricas
- CRUD completo de categorías
- CRUD completo de tareas
- Relación entre tareas y categorías
- Interfaz responsiva
